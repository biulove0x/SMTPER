# SMTPER
Testing smtp server like smtper.net


### Setup
settings/smtp.ini
```
[smtp_server]
HOST = webmail.host.com
PORT = 587
USER = username
PASS = password
FROM = noreply@host.com
```

### Requirement
```
$ python3 -m pip install email
$ python3 -m pip install smtplib
```

### How to run
```
$ python3 app.py
#################################################
# @author : biulove0x                           #
# @name   : SMTP server tester like smtper.net  #
#################################################
 [ + ] emailreceived@mail.com
```
